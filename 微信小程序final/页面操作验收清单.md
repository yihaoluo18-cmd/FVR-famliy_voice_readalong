# 页面操作验收清单（可勾选）

说明：以下按真实页面操作顺序设计，可用于开发者工具/真机逐项验收。每一步包含预期现象与结果记录位。

## A. 魔法声音克隆室（含音色管理入口）

- [ ] A1 进入“魔法声音克隆室”页面
  - 操作：从底部“声音克隆”进入。
  - 预期：页面正常加载，顶部标题“魔法声音克隆室”，无报错弹窗。
  - 结果：未通过（需真机点击确认页面视觉与交互）

- [ ] A2 检查页面内“管理已有音色（支持删除）”入口
  - 操作：在克隆室页点击“管理已有音色（支持删除）”。
  - 预期：跳转到“音色管理”页。
  - 结果：已通过（代码检查：克隆室已接入管理入口）

- [ ] A3 检查左上角全局音色下拉中的管理入口
  - 操作：在任一带左上角音色组件页面（如 AI 魔法绘本、小小演说家）展开音色下拉。
  - 预期：出现“管理音色（可删除）”项，点击后进入同一个“音色管理”页。
  - 结果：已通过（代码检查：左上角音色下拉已接入同一管理页）

- [ ] A4 音色管理页切换音色
  - 操作：点击任一非默认音色条目。
  - 预期：提示切换成功；返回其他页面后左上角当前音色同步更新。
  - 结果：未通过（需真机点击验证切换提示与跨页同步）

- [ ] A5 音色管理页删除音色（非默认）
  - 操作：点击某非默认音色“删除”并确认。
  - 预期：删除成功提示；列表刷新后该音色消失。
  - 结果：未通过（需真机点击并联调后端删除接口）

- [ ] A6 默认音色保护
  - 操作：尝试删除“基础音色（默认）”。
  - 预期：提示“基础音色不可删除”。
  - 结果：已通过（代码检查：默认音色删除前端已拦截）

- [ ] A7 克隆流程恢复提示
  - 操作：在克隆室进行到第2或第3步后退出，再进入。
  - 预期：弹出“继续上次克隆进度？”；点“继续”恢复步骤，点“重新开始”清空进度。
  - 结果：未通过（需真机执行离开/返回流程验证弹窗）

- [ ] A8 重名覆盖确认
  - 操作：使用已存在音色名称启动训练。
  - 预期：弹出“名称已存在，是否覆盖旧音色”；取消不启动，确认后继续训练。
  - 结果：未通过（需真机+训练数据验证重名覆盖对话）

- [ ] A9 训练中科普讲解
  - 操作：启动训练进入第3步。
  - 预期：显示科普卡片（吉祥物+科普文案），并随训练阶段轮播；有播报状态动画。
  - 结果：未通过（需真机发起训练观察科普轮播播报）
