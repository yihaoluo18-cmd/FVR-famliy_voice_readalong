# 小小演说家：配图生图提示词（100题）

说明：
- 每题提供：**主提示词（Prompt）**、**负面提示词（Negative）**、**可选细化（Options）**。
- 统一风格建议：**儿童绘本风、扁平插画、低细节、柔和马卡龙配色、线条清晰、主体居中、留白充足、无文字无水印**。
- 推荐尺寸：\(1024\times1024\) 或 \(768\times768\)。

通用 Negative（可直接复制到所有题）：  
`文字, 字母, 水印, logo, 二维码, 真实照片, 写实, 过度细节, 恐怖, 血腥, 暴力, 武器, 成人, 香烟, 酒精, 低清晰度, 模糊`

---
